Vigenère Cipher

The Vigenère Cipher uses a keyword to shift letters in the plaintext by varying amounts. Each letter in the keyword is used to determine the shift for the corresponding letter in the plaintext.

Example:

Plaintext: HELLO

Keyword: KEY

Vigenère Cipher: RIJVS

Explanation: The letters in the keyword (KEY) shift each plaintext letter: H -> R, E -> I, L -> J, L -> V, and O -> S.
