Affine Cipher

The Affine Cipher uses a mathematical function (ax + b) mod m to encrypt letters. The multiplicative key 'a' must be coprime with the alphabet size (26 for English), and the additive key 'b' can be any number.

Example:

Plaintext: HELLO

Multiplicative key (a) = 5, Additive key (b) = 8

Ciphertext: RCLLA

Explanation: Using the Affine Cipher formula, each letter is encrypted by applying the formula (5 * x + 8) mod 26.
